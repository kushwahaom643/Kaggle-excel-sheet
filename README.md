# CT&DT-Galgotias-Batch No:5-Task 1-Product Failure Analysis

## Dataset

Machine Failure Analysis Dataset

## Problem Statement

Analyze machine parameters to identify factors leading to machine failure and propose solutions to reduce failures.

## Key Failure Factors

1. High Air Temperature
2. High Process Temperature
3. Excessive Rotational Speed
4. High Torque
5. Tool Wear

## Proposed Solution

Implementation of a Smart Predictive Maintenance System using real-time sensor monitoring and machine learning-based failure prediction.

## Deliverables

* Excel Sheet containing analyzed machine data
* Failure causes and recommendations
* Product improvement suggestions
